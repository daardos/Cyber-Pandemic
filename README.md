# 🌐 CyberPandemic: The Last Citadel 🚀

**CyberPandemic** — это высокотехнологичный интерактивный симулятор глобальной эпидемии в стиле **Cyberpunk**. Проект построен на чистом JavaScript и демонстрирует работу сложных системных алгоритмов, процедурной генерации и искусственного интеллекта юнитов.

---

## 💎 Ключевые особенности (Hardcore JS)

Проект реализует глубокую логику взаимодействия объектов, имитирующую реальные биологические и логистические процессы:

*   **🛡️ Система Bio-Resistance (3-Hit Rule):** Юниты не заражаются мгновенно. Реализован счетчик контактов — только после **3-го столкновения** с вирусом объект меняет статус. 
*   **⏱️ Internal Cooldown (ICD):** После каждого контакта юнит получает временный "иммунный щит" на 3 секунды (визуальное белое кольцо). Это предотвращает экспоненциальный взрыв заражения и добавляет стратегии.
*   **🚢 Автономная логистика (Ferry System):** Корабли-торговцы автоматически перевозят жителей между островами. Корабли являются **неизлечимыми носителями** — один зараженный борт может погубить весь архипелаг.
*   **☠️ ИИ Одинокого Пирата:** Уникальный юнит-преследователь. Он сканирует карту, находит торговые суда и атакует их, распространяя инфекцию. Имеет механику "форсажа" при клике игрока.
*   **🚀 Орбитальный удар (DEFCON 1):** Система мониторинга в реальном времени. Если остров признан полностью потерянным (100% заражения), JS вызывает тактическую ракету для точечной дезинфекции (взрыв в радиусе 150px).
*   **🏆 Протокол «Цитадель»:** Финальный этап выживания. При уничтожении большинства городов в центре генерируется Мега-остров с 10-местным супер-госпиталем и ускоренным лечением.

---

## 🛠 Технический стек

*   **Core:** Vanilla JavaScript (ES6+).
*   **Graphics:** HTML5 Canvas API (стабильные 60 FPS при 200+ активных объектах).
*   **Math:** Векторная физика (Vector velocity), тригонометрические расчеты для круговых границ островов, линейная интерполяция для полета ракет.
*   **State Management:** Полностью контролируемый жизненный цикл каждого объекта (Healthy -> Infected -> Healing -> Dead).
